## Tech Stack

- Vue 3 with Composition API
- Vite for fast development and building
- TailwindCSS for styling
- Lucide Vue Next for icons
- QRCode library for QR code generation

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
